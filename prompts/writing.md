# Writing & Communication Prompts

A collection of prompts for creative writing, professional communication, content creation, and storytelling.

---

## Blog & Content Writing

### Write a Blog Post

```
Write a blog post titled "[title]" for an audience of [target audience].

The post should:
- Be approximately [word count] words
- Have an engaging introduction that hooks the reader
- Cover the following key points: [point 1], [point 2], [point 3]
- Include a clear call to action at the end
- Use a [formal / conversational / technical] tone
```

### Write a LinkedIn Article

```
Write a LinkedIn article about [topic] aimed at [target audience, e.g., "software engineers", "marketing professionals"].

Make it:
- Around [500–800] words
- Start with a compelling hook or personal anecdote
- Include 3–5 actionable takeaways
- End with a thought-provoking question to encourage comments
- Professional yet conversational in tone
```

### Improve Existing Content

```
Rewrite the following text to make it clearer, more engaging, and better structured:

[paste your text here]

Target audience: [describe your audience]
Desired tone: [professional / friendly / persuasive / etc.]
Main goal of the content: [inform / sell / entertain / etc.]

Preserve the key message but improve readability and impact.
```

## Professional Email

### Draft a Professional Email

```
Write a professional email with the following details:

To: [recipient's role or name]
Subject: [email subject]
Purpose: [what you need to communicate or request]
Key points to include:
- [point 1]
- [point 2]
Desired tone: [formal / friendly / assertive]

Keep it concise and clear.
```

### Write a Follow-Up Email

```
Write a follow-up email to [recipient description] after [context, e.g., "a job interview", "a sales call", "a meeting"].

Date of original interaction: [date]
Key topics discussed: [summarize briefly]
Goal of the follow-up: [e.g., express continued interest, confirm next steps, ask a question]

Keep it short, professional, and genuine.
```

### Decline Politely

```
Write a polite but firm email declining [request or offer description].

Reason for declining (optional to mention): [reason]
Relationship with recipient: [colleague / client / vendor / etc.]

Maintain a positive tone and leave the door open for future collaboration if appropriate.
```

## Creative Writing

### Start a Short Story

```
Write the opening scene (approximately [word count] words) of a short story with the following elements:

Genre: [genre, e.g., science fiction, mystery, literary fiction]
Setting: [time and place]
Main character: [brief description]
Opening situation: [what is happening as the story begins]
Mood/tone: [e.g., tense, melancholic, whimsical]

End the scene with a hook that makes the reader want to continue.
```

### Develop a Character

```
Create a detailed character profile for a fictional character with the following basics:

Name: [name]
Role in story: [protagonist / antagonist / supporting]
Genre: [genre]

Include:
- Physical description
- Personality traits (strengths and flaws)
- Background and backstory
- Core motivation and goal
- Greatest fear
- Speech patterns or mannerisms
- How they change or could change through the story
```

### Write Dialogue

```
Write a dialogue scene between [character A description] and [character B description].

Context: [what has just happened or what is at stake]
Goal of the scene: [what needs to be established, e.g., tension, information, relationship development]
Tone: [e.g., tense, humorous, emotional]

Make each character's voice distinct and advance the plot or reveal character through subtext.
```

## Marketing & Persuasion

### Write a Product Description

```
Write a compelling product description for [product name].

Product details:
- What it does: [functionality]
- Key features: [feature 1], [feature 2], [feature 3]
- Target customer: [describe ideal buyer]
- Unique selling point: [what makes it stand out]

Length: [1 short paragraph / 150–200 words]
Tone: [energetic / professional / minimalist]
```

### Write a Social Media Post

```
Write [number] social media post(s) for [platform, e.g., Twitter/X, Instagram, Facebook] about [topic or product].

Goal: [raise awareness / drive engagement / promote a product / share a tip]
Target audience: [describe]
Tone: [fun / professional / inspirational]
Include: [a call to action / relevant hashtags / an emoji or two]
Character/word limit: [if applicable]
```

### Write a Call to Action

```
Write [number] variations of a call-to-action (CTA) for [context, e.g., a landing page, an email, a button].

The desired action: [what you want the reader to do]
Audience motivation: [what benefit drives them]
Tone options to try: [urgent / friendly / benefit-focused]

Make them concise, action-oriented, and compelling.
```

## Editing & Proofreading

### Proofread Text

```
Proofread the following text and correct all grammar, spelling, and punctuation errors. Do not change the meaning, style, or structure.

[paste your text here]

After corrections, provide a brief summary of the types of errors found.
```

### Adjust Tone

```
Rewrite the following text to match a [target tone, e.g., "more formal", "warmer and more empathetic", "more concise and direct"] tone:

[paste your text here]

Preserve the original meaning and key information.
```

### Simplify Complex Text

```
Simplify the following text so it can be understood by someone with no background in [field]:

[paste your text here]

Avoid jargon. Use short sentences and everyday language. Aim for a reading level of approximately [grade level or "general adult audience"].
```
