# Analysis & Research Prompts

A collection of prompts for data analysis, research synthesis, critical thinking, and decision-making.

---

## Data Analysis

### Analyze a Dataset

```
I have a dataset with the following structure:

Columns: [list column names and types, e.g., "date (datetime), revenue (float), region (string)"]
Number of rows: approximately [N]
Source: [where the data comes from]

Please help me:
1. Identify key trends and patterns
2. Highlight any anomalies or outliers
3. Suggest the most meaningful metrics to track
4. Recommend visualizations that would best communicate the insights
```

### Interpret Results

```
I ran [type of analysis, e.g., "a linear regression", "an A/B test", "a survey"] and got the following results:

[paste or describe your results]

Please:
1. Explain what these results mean in plain language
2. Identify what is statistically significant vs. potentially noise
3. Describe the practical implications
4. Point out any limitations or caveats I should be aware of
```

### Write a Data Analysis Plan

```
I want to analyze [describe the problem or question you want to answer] using data from [data source].

Help me create a structured analysis plan that includes:
1. Clearly stated research questions or hypotheses
2. Data cleaning and preparation steps
3. Analysis methods to use
4. Key metrics and KPIs to measure
5. How to present and communicate findings
```

## Research Synthesis

### Summarize a Topic

```
Provide a comprehensive summary of [topic].

Include:
- A brief overview of the subject
- Key concepts and definitions
- Major schools of thought or perspectives
- Important findings or milestones
- Current state of knowledge and open questions

Target audience: [experts / general readers / students]
Desired length: [brief overview / detailed summary]
```

### Compare and Contrast

```
Compare and contrast [subject A] and [subject B] across the following dimensions:
- [dimension 1, e.g., approach, cost, scalability]
- [dimension 2]
- [dimension 3]

Provide a balanced analysis that highlights the strengths and weaknesses of each. Conclude with a recommendation for [specific use case or context].
```

### Literature Review Summary

```
I am researching [topic]. Below are summaries or excerpts from several sources:

Source 1: [paste or describe]
Source 2: [paste or describe]
Source 3: [paste or describe]

Please synthesize these sources by:
1. Identifying common themes and agreements
2. Noting areas of disagreement or debate
3. Identifying gaps in the current knowledge
4. Summarizing the overall state of evidence on this topic
```

## Critical Thinking

### Devil's Advocate Analysis

```
I am considering the following position or plan:

[describe your position, idea, or plan]

Please argue the strongest possible case AGAINST this position. Identify weaknesses, risks, unintended consequences, and assumptions that may not hold. Be thorough and direct.
```

### Identify Logical Fallacies

```
Review the following argument and identify any logical fallacies, weak reasoning, or unsupported claims:

[paste the argument or text]

For each issue found:
1. Name the type of fallacy or weakness
2. Quote the specific part of the argument where it appears
3. Explain why it is problematic
4. Suggest how the argument could be strengthened
```

### Challenge Assumptions

```
I am planning to [describe your plan or decision].

My key assumptions are:
1. [assumption 1]
2. [assumption 2]
3. [assumption 3]

For each assumption, please:
- Evaluate whether it is well-founded or potentially flawed
- Identify what evidence would support or disprove it
- Suggest what would happen to the plan if the assumption turned out to be wrong
```

## Decision Making

### Structured Decision Analysis

```
I need to make a decision about [decision description].

My options are:
1. [option 1]
2. [option 2]
3. [option 3, if applicable]

My criteria for a good decision (in rough order of importance):
1. [criterion 1]
2. [criterion 2]
3. [criterion 3]

My constraints: [time / budget / resources / etc.]

Please help me systematically evaluate each option against my criteria and recommend the best choice with reasoning.
```

### Risk Analysis

```
I am planning to [describe plan or decision].

Please conduct a risk analysis:
1. Identify the top [5–10] potential risks
2. For each risk, estimate:
   - Likelihood (low / medium / high)
   - Impact (low / medium / high)
3. Prioritize the risks by overall severity
4. Suggest mitigation strategies for the highest-priority risks
```

### Pre-Mortem Analysis

```
I am about to [describe a project, plan, or decision].

Imagine it is [X months/years] in the future and this has failed completely. Walk me through:
1. The most likely reasons it failed
2. Warning signs that should have been heeded earlier
3. Actions I can take now to prevent the most probable failure scenarios
```

## Competitive & Market Analysis

### Competitor Analysis

```
Analyze [competitor name or type] as a competitor to [my product/company description].

Cover:
1. Their product/service offering and key features
2. Target market and customer segments
3. Pricing strategy
4. Strengths and advantages
5. Weaknesses and vulnerabilities
6. How they position themselves vs. the market

Conclude with strategic implications for how I should differentiate or compete.
```

### SWOT Analysis

```
Perform a SWOT analysis for [company / product / project / idea]:

Context: [brief description]

Provide a thorough analysis of:
- Strengths (internal advantages)
- Weaknesses (internal disadvantages)
- Opportunities (external factors to exploit)
- Threats (external risks to address)

After the SWOT, suggest 2–3 strategic actions based on the analysis.
```
