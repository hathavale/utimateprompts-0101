# Productivity & Task Management Prompts

A collection of prompts to help with planning, goal setting, prioritization, and getting more done.

---

## Planning & Goal Setting

### Create a Project Plan

```
Help me create a project plan for [project description].

Goal: [what success looks like]
Timeline: [start date] to [end date]
Team: [number of people / roles involved]
Key constraints: [budget / resources / dependencies]

Break the project into phases with:
- Milestones and deliverables for each phase
- Estimated time for each task
- Dependencies between tasks
- Potential risks and contingencies
```

### Define SMART Goals

```
Help me turn the following vague goal into a set of SMART goals (Specific, Measurable, Achievable, Relevant, Time-bound):

Vague goal: [describe what you want to achieve]

Context:
- Available time/resources: [describe]
- Why this matters: [your motivation]

For each SMART goal, also suggest 1–2 leading indicators I can track weekly to stay on course.
```

### Quarterly Planning

```
Help me plan my next quarter. My role is [job title/description].

My top priorities for the quarter are:
1. [priority 1]
2. [priority 2]
3. [priority 3]

My constraints: [time, team size, budget, etc.]

Help me:
1. Break each priority into concrete monthly milestones
2. Identify potential conflicts or resource bottlenecks
3. Suggest how to allocate my time across priorities
4. Define what "done" looks like for each priority
```

## Prioritization

### Prioritize a Task List

```
I have the following list of tasks and need help prioritizing them:

[paste your task list here]

Context:
- My main goal right now: [describe]
- Deadline pressures: [any hard deadlines]
- My energy levels: [when I'm most productive]

Please organize these tasks by priority using the [Eisenhower Matrix / MoSCoW method / impact vs. effort], and suggest which to tackle first today.
```

### Evaluate What to Cut

```
I am overcommitted and need to reduce my workload. Here are my current commitments:

[list your commitments]

My core responsibilities that cannot be cut: [list them]

Help me identify:
1. Commitments I can drop entirely
2. Commitments I can delegate
3. Commitments I can defer to a later date
4. How to communicate each cut professionally
```

## Brainstorming

### Generate Ideas

```
Generate [number] creative ideas for [topic or problem].

Context: [provide any relevant constraints, goals, or audience information]
Desired qualities: [e.g., low cost, fast to implement, high impact, novel]

For each idea, provide:
- A one-sentence description
- The main benefit
- A potential drawback or risk
```

### Overcome a Creative Block

```
I am stuck on [describe what you are working on and where you are stuck].

What I have tried so far: [describe]
What isn't working: [describe the block]

Help me:
1. Approach this from 3 completely different angles
2. Ask 5 questions that might unlock a new perspective
3. Suggest one unconventional or counterintuitive approach I probably haven't considered
```

### Reverse Brainstorm

```
Instead of asking "How do I solve [problem]?", help me reverse-brainstorm:

"How could I make [problem] as bad as possible?"

List [10–15] ways to make the problem worse. Then, for each item, flip it to identify a potential solution or improvement I might have overlooked.
```

## Meeting & Communication

### Prepare for a Meeting

```
Help me prepare for a [meeting type, e.g., "1-on-1 with my manager", "client kickoff", "team retrospective"] meeting.

Topic / agenda: [what will be discussed]
My goals for the meeting: [what I want to achieve]
Attendees: [who will be there]
Duration: [how long]

Please provide:
1. Key questions I should ask
2. Points I should be prepared to discuss
3. Potential objections or difficult topics and how to handle them
4. A suggested meeting agenda
```

### Write a Meeting Agenda

```
Create a focused meeting agenda for a [duration]-minute meeting on [topic].

Attendees: [list roles]
Goal of the meeting: [desired outcome]
Key topics to cover:
1. [topic 1]
2. [topic 2]
3. [topic 3]

Format as a timed agenda with clear owners for each item and the expected outcome for the meeting.
```

### Write a Status Update

```
Write a concise status update for [project or initiative name] to share with [audience, e.g., "my manager", "the team", "stakeholders"].

Current status: [on track / at risk / blocked]
Accomplishments since last update:
- [accomplishment 1]
- [accomplishment 2]

In progress:
- [item 1]
- [item 2]

Blockers or risks: [any issues]
Next steps: [what's coming next]

Keep it concise and use a [bullet-point / narrative] format.
```

## Personal Productivity

### Design a Daily Routine

```
Help me design an ideal daily routine for a [role description, e.g., "remote software engineer", "freelance writer"].

My constraints:
- Work hours: [preferred hours]
- Energy pattern: [e.g., "most focused in the morning"]
- Non-negotiables: [family time, exercise, etc.]
- Key work tasks: [what I need to accomplish each day]

Provide a sample hourly schedule with explanations for the structure and tips for sticking to it.
```

### Beat Procrastination

```
I have been procrastinating on [task or project description].

Reason I think I'm procrastinating: [e.g., "it feels overwhelming", "I'm not sure where to start", "I don't find it interesting"]

Help me:
1. Break it into the smallest possible next steps (no step should take more than 30 minutes)
2. Identify and remove the first obstacle
3. Suggest one technique to build momentum and get started today
```

### Weekly Review Template

```
Guide me through a weekly review. Help me reflect on the past week and plan the next one.

Past week:
- What I accomplished: [list]
- What I didn't finish: [list]
- What went well: [list]
- What I'd do differently: [list]

For the coming week, help me:
1. Identify my top 3 priorities
2. Schedule time blocks for deep work
3. Note any potential obstacles and how to handle them
4. Define what a successful week looks like
```
