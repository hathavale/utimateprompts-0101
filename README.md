# Ultimate Prompts Collection

A curated collection of high-quality prompts for AI assistants, organized by category.

## Table of Contents

- [Coding & Development](prompts/coding.md)
- [Writing & Communication](prompts/writing.md)
- [Analysis & Research](prompts/analysis.md)
- [Productivity & Task Management](prompts/productivity.md)
- [Learning & Education](prompts/learning.md)

## How to Use

1. Browse the category that matches your task.
2. Copy the prompt you want to use.
3. Replace any `[bracketed placeholders]` with your specific details.
4. Paste the prompt into your AI assistant.

## Categories

| Category | Description |
|---|---|
| [Coding](prompts/coding.md) | Code generation, debugging, code review, and software architecture |
| [Writing](prompts/writing.md) | Blog posts, emails, stories, and professional documents |

## Contributing

Contributions are welcome! To add a new prompt:

1. Fork this repository.
2. Add your prompt to the relevant category file in the `prompts/` directory.
3. Follow the existing format (title, prompt text, optional notes).
4. Submit a pull request.
